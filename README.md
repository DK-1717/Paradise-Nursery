# Paradise Nursery Shopping Application

## Project Description
Paradise Nursery is an online plant shop where users can browse plants and manage a shopping cart.

## Features
- Browse plants by category
- Add plants to cart
- Increase/decrease quantity
- Remove items
- Dynamic total price
- Responsive UI

## Technologies Used
- React
- Redux Toolkit
- React Router
- CSS

## How to Run
npm install
npm run dev
